# Step2: Attach Public IP to VM

Azure Console -> Virtual Machines -> Open required VMs -> Scroll down and open 'Networking' -> Open 'Public IP address' by clicking on 'Configure' -> Edit IP configuration by clicking on 'ipconfigxfusion-vm-pip' -> Select 'Associate public IP address' -> Select 'Public IP address' -> Save
