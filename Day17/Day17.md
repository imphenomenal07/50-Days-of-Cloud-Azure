# Day 17: Create a Public Azure Blob Storage Container

# Step1: Login into Azure Console

# Step2: Create Storage Account

Azure Console ➡️ Storage Account ➡️ Create
#Fill Details ➡️ Review + Create

# Step3: Enable Blob anonymous access

➡️Once Account is created, click on 'Go to resources' ➡️ Settings ➡️ Configuration ➡️ Allow Blob anonymous access

Enable and Save!!

# Step4: Create Public Blob Container

Storage Account ➡️ Storage Browser ➡️ Blob Contianers ➡️ Add Container

Name:
Anonymous access level: Select 'Container (anonymous read access for containers and blobs'

#If 'Anonymous access level' option is grayed out, refresh the console page once!!
