# Day 9: Attach Network Interface Card (NIC) to Azure Virtual Machine

# Step1: Login into Azure Console

# Step2: Attach Network Interface (NIC) 

Azure Console -> Virtual Machines -> Open required VMs -> First 'STOP' VM

Scroll down and open 'Networking' -> Attach Network Interface -> Select from existing NIC -> OK

Wait till it is attached!
